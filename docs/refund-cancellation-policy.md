# Mibu 退款與取消政策

## 政策概覽

本政策適用於透過 Mibu 官方網站購買的所有商家訂閱服務。

---

## 一、取消政策

### 1.1 隨時取消
- 您可以**隨時**取消訂閱，無需支付額外費用
- 取消後，您的訂閱將持續有效至**當期結束**
- 當期結束後，將自動停止扣款

### 1.2 取消方式
1. 登入商家後台
2. 前往「訂閱管理」頁面
3. 點擊「取消訂閱」按鈕
4. 確認取消

### 1.3 取消後的權益
- 取消立即生效，但服務持續至當期結束
- 當期結束前仍可使用所有已付費功能
- 資料將保留 30 天，之後刪除

---

## 二、退款政策

### 2.1 七天鑑賞期（符合台灣消費者保護法）

根據台灣《消費者保護法》第 19 條，線上購買之服務享有 **7 天鑑賞期**：

| 條件 | 說明 |
|------|------|
| 適用對象 | 首次訂閱的 B2C 消費者 |
| 鑑賞期 | 首次付款後 7 個日曆天內 |
| 退款金額 | 全額退款 |
| 退款方式 | 原支付方式退回 |
| 處理時間 | 5-10 個工作天 |

### 2.2 退款條件

**可申請退款的情況：**
- ✅ 首次付款後 7 天內申請
- ✅ 帳戶未大量使用服務功能
- ✅ 技術問題導致無法使用服務（需提供證明）

**不適用退款的情況：**
- ❌ 超過 7 天鑑賞期
- ❌ 已大量使用服務功能（如：建立超過 3 個優惠券）
- ❌ 續訂期間（非首次訂閱）
- ❌ 企業 B2B 訂閱（需另議）

### 2.3 退款申請方式

1. **Email 申請**：發送郵件至 support@mibu-travel.com
   - 主旨：[退款申請] 您的商家名稱
   - 內容：訂閱帳號、退款原因、付款日期

2. **客服表單**：填寫官網客服表單

### 2.4 退款處理流程

```
申請提交 → 資格審核 (1-2 工作天) → 退款處理 (3-5 工作天) → 款項退回
```

## 三、年訂閱特別條款

### 3.1 年訂閱取消
- 年訂閱可隨時取消，取消後享有至當年度結束
- 取消不會立即終止服務

### 3.2 年訂閱退款
- **7 天內**：全額退款
- **7 天後**：不提供退款，但服務持續至當年度結束
- **特殊情況**：經審核可酌情提供按比例退款

---

## 四、付款失敗處理

### 4.1 自動重試
- 付款失敗後，系統將在 3 天內重試
- 重試 3 次仍失敗，訂閱將自動暫停

### 4.2 暫停與恢復
- 暫停期間無法使用付費功能
- 更新付款方式後可立即恢復
- 暫停超過 30 天，訂閱將自動取消

## 附錄：法規依據

### 台灣消費者保護法第 19 條

> 通訊交易或訪問交易之消費者，得於收受商品或接受服務後七日內，以退回商品或書面通知方式解除契約，無須說明理由及負擔任何費用或對價。
